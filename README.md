# azureapp
simple Webchat using Ajax php. 
By Ikhsanhaw and my Tutor John.
### Uses
* AngularJS 
* Bootstrap 
* PHP

### Features
* New message detector
  * Desktop notifications like WhatsApp web platform
  * Sound notifications
  * Title change notification
* Online users count (with ip detection)
* Shortcodes to send links and images
* Admin user (with a secret nickname prefix)
  * List online clients IP (By admin command "/online")
  * Clear the chat history (By admin command "/clear")
* admin nickname is "adminpass123"
